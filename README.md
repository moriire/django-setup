Django-setup is a linux based tool, that
automates the boring setup process of django
projects.

## Installation

```sh

~ $ git clone https: //github.com/moriire/django-setup.git
```
```sh
~ $ cd django-setup
```
```sh
~/django-setup $ chmod u+x ./django-setup.sh
```
```sh
~ $ django-setup
```
